# ArdiA Health Labs Presentation Design Spec

## Visual Style
- **Theme**: Dark Mode Professional
- **Background**: #121212 (Deep Black/Gray) with subtle gradient overlays
- **Accent**: #8BAFD0 (Medical/Tech Blue)
- **Typography**: Montserrat (Headings) + Inter (Body)

## Slide Layouts

### 1. Title Slide
- Centered content
- Large logo placeholder
- Hero text
- Founder contact info

### 2. Content Slide (Left/Right)
- 40/60 or 50/50 split
- Text on left, Visual on right
- Bullet points with custom icons

### 3. Diagram Slide (Architecture/Process)
- Full width diagram area
- CSS/SVG drawn shapes for:
  - Architecture (Data -> Engine -> Output)
  - Process Loop (Collect -> Analyze -> Predict -> Intervene -> Learn)

### 4. Card Grid (Stakeholders/Market)
- 3-column or 2x2 grid
- Cards with subtle borders (#2D2D2D) and hover effects
- Icon + Title + Metric + Description

### 5. Data/Timeline
- Horizontal timeline flow
- Big bold metrics ($8.1M) with trend indicators

## Common Elements
- **Navbar/Footer**: Minimal branding on bottom left/right
- **Icons**: Lucide Icons (CDN)
- **Charts**: Chart.js (CDN) if needed, or CSS stats
